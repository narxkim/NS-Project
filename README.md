# NS-Project 🧲

> 자석처럼 목표 객체를 정확하게 탐지하는 게임 캐릭터 객체 탐지(Object Detection) 프로젝트

---

## 📌 프로젝트 개요

게임 환경 내 특정 캐릭터를 탐지하기 위한 Object Detection 모델 구축 프로젝트입니다.

게임 스크린샷 데이터를 기반으로 특정 캐릭터를 탐지하며, 객체 탐지 기술을 활용한 게임 플레이 자동화 가능성을 검증하는 것을 목표로 합니다.

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 게임 환경 내 특정 캐릭터 객체 탐지(Object Detection) 모델 구축 |
| 팀명 | 자석(NS) |
| 기간 | 2026.06.10 ~ 2026.06.17 |
| 프로젝트 유형 | Computer Vision · Object Detection |
| 개발 환경 | Python · YOLO |

---

## 🎯 프로젝트 목표

- 게임 화면(스크린샷)을 입력으로 받아 특정 캐릭터 탐지
- Object Detection 기반 캐릭터 위치 인식
- 다양한 게임 환경(전투, 이펙트, UI 가림 등)에 대한 탐지 성능 확보
- 게임 플레이 자동화에 활용 가능한 객체 탐지 모델 구축
